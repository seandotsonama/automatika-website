Axis A Advance - Cylinder Control

Automatika Ladder Logic Example | Rockwell ControlLogix Style

Program: Axis_A_Advance

Rung 0 Safety Interlock
XIC(FC.0) XIO(A_Advance.Fault) XIC(B_Retract.InPos) → OTE(A_Advance.Safe) — All conditions TRUE, output energized (green)
Rung 1 Trigger Latch (Always False)
AFI → OTL(A_Advance.Trig) — Always False Instruction prevents latch, rung never energized
Rung 2 Auto/Manual Command with Seal-In
Branch 1: XIC(AutoMode) XIC(A_Advance.Trig) | Branch 2: XIC(ManualMode) XIC(A_Advance.PB) → XIC(A_Advance.Safe) XIO(A_Advance.InPos) OTE(A_Advance.Cmd) with seal-in via XIC(A_Advance.Cmd)
Rung 3 Position Detection
Branch 1: XIC(A_ADV_INPUT) XIO(A_RET_INPUT) | Branch 2: XIO(A_ADV_INPUT) XIC(A_RET_INPUT) → OTE(A_Advance.InPos) / OTE(A_Advance.Retracted)
Rung 4 Fault Detection Timer
XIC(A_Advance.Cmd) XIO(A_Advance.InPos) → TON(A_Advance.FaultTMR, 5000ms) with seal-in via XIC(A_Advance.Fault) → XIO(FaultTMR.DN) OTE(A_Advance.Fault)
