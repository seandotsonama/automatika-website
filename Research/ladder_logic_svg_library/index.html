Bit (Relay) Instructions - Contacts

XIC - Examine If Closed
Normally Open | TRUE when bit = 1
XIC - Energized
Power flowing through contact
XIO - Examine If Open
Normally Closed | TRUE when bit = 0
XIO - Passing
Bit = 0, contact allows power flow

Bit (Relay) Instructions - Coils

OTE - Output Energize
Non-retentive output coil
OTE - Energized
Output active, power flowing
OTL - Output Latch
Retentive latch, requires OTU to clear
OTU - Output Unlatch
Clears latched output

One-Shot Instructions

ONS - One Shot
True for 1 scan on rising edge
OSR - One Shot Rising
Output bit set for 1 scan on rising edge
OSF - One Shot Falling
Output bit set for 1 scan on falling edge

Timer Instructions

TON - Timer On Delay
Times while rung true, resets on false
TON - Active (Timing)
EN energized, timing in progress
TOF - Timer Off Delay
Times after rung goes false
RTO - Retentive Timer
Retains ACC, requires RES to reset

Counter Instructions

CTU - Count Up
Increments ACC on rising edge
CTD - Count Down
Decrements ACC on rising edge
RES - Reset
Resets timer/counter ACC to 0

Comparison Instructions

EQU - Equal
True when A = B
NEQ - Not Equal
True when A != B
GRT - Greater Than
True when A > B
GEQ - Greater or Equal
True when A >= B
LES - Less Than
True when A < B
LEQ - Less or Equal
True when A <= B

Math & Move Instructions

ADD - Add
Dest = A + B
SUB - Subtract
Dest = A - B
MUL - Multiply
Dest = A * B
DIV - Divide
Dest = A / B
MOV - Move
Copies Source to Dest
CPT - Compute
Free-form expression evaluation

Program Control

JSR - Jump to Subroutine
Calls routine with parameters
SQO - Sequencer Output
Steps through output array

Structural Elements

Power Rails
Left (source) and Right (return) rails
Branch / Parallel (OR Logic)
Seal-in circuit: (Start OR Running) AND NOT Stop = Motor
Complete Rung Example
XIC(AutoMode) GEQ(Temp,70) OTE(Fan)
